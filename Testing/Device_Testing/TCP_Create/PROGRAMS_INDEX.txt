USAGE
===============================================================================

1. Choose program based on number of registers needed (5-50)
2. Ensure gateway is in BLE Development mode
3. Run program:

   python create_device_10_registers.py

   (Replace 10 with desired register count: 5, 15, 20, 25, 30, 35, 40, 45, 50)

4. Wait for device and registers creation
5. Verify on gateway that device ID is captured
6. Test with Modbus slave simulator

===============================================================================
TESTING WORKFLOW
===============================================================================

Step 1: Start Modbus Slave Simulator
        cd ../Modbus_Slave_Simulator
        python modbus_slave_5_registers.py

Step 2: Run Device Creation Program
        cd ../Device_Testing
        python create_device_15_registers.py  (example)

Step 3: Verify Connection
        - Check simulator logs for gateway connection
        - Verify gateway is polling registers
        - Monitor data updates

Step 4: Test Data Flow
        - Simulator updates every 10 seconds
        - Gateway polls every 1 second
        - Data should flow: Simulator -> Gateway -> MQTT/HTTP

===============================================================================
TROUBLESHOOTING
===============================================================================

Issue: BLE device not found
Fix:   - Check gateway is powered on
       - Verify BLE Development mode is active
       - LED should be ON (steady)

Issue: Device creation fails
Fix:   - Check JSON payload in debug output
       - Verify gateway firmware version (v2.2.0)
       - Check BLE connection stability

Issue: Register creation fails
Fix:   - Ensure device ID was captured
       - Check address range (0 to N-1)
       - Verify no duplicate register names

Issue: Too many registers
Fix:   - Use smaller program (e.g., 10 or 15 registers)
       - Check gateway memory capacity
       - Monitor gateway performance
